## Get Started

<Steps>
<Step title="Quick Installation">
  Run our one-command installer to deploy Citizen with production-ready configuration.
  
  ```bash
  curl -sSL https://raw.githubusercontent.com/citizenteam/citizen/main/install.sh | bash
  ```
  
  <Check>
  Installation typically completes in 2-3 minutes with automatic SSL setup.
  </Check>
</Step>

<Step title="Configure DNS">
  Point your domain to your server for automatic SSL certificate generation.
  
  ```bash
  # Required DNS records
  yourdomain.com      A     YOUR_SERVER_IP
  *.yourdomain.com    A     YOUR_SERVER_IP
  ```
  
  <Warning>
  DNS propagation can take up to 48 hours. SSL certificates generate automatically after DNS is configured.
  </Warning>
</Step>

<Step title="Access Dashboard">
  Navigate to your domain and complete the initial admin user setup.
  
  <Check>
  Your Citizen dashboard is now ready for application management!
  </Check>
</Step>
</Steps>
